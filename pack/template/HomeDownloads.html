About Preview

Packfire Framework 1.0-elenor and 1.0-sofia are preview versions of the framework. There may be features which are not fully implemented, unstable, or not suitable for older versions of PHP.

Preview is here.

1.0-elenor is a great start for you to take a peek into the power and clarity of Packfire Framework. You can use the preview to work on small projects that demands rapid development or prototyping.

Git Access

You can fetch the framework directly from Github by running the following git command:

$ git clone --recursive https://github.com/packfire/framework.git packfire

Note that fetching the latest work-in-progress version from the Git repository may also bring bugs along too.
